## 4. 已核实的技术 API(照这个写,别凭记忆)

**FastMCP**:
- `from fastmcp import FastMCP`;`mcp = FastMCP("name")`
- `@mcp.tool()` / `@mcp.resource("uri")` / `@mcp.prompt()`(docstring 当 description)
- resource 模板:`@mcp.resource("notes://note/{title}")` + 路径安全 `(root/title).resolve().is_relative_to(root)`
- 传输:`mcp.run(transport="stdio")` 或 `mcp.run(transport="http", host, port)`(端点 `/mcp/`);`@mcp.custom_route("/health")` 健康检查
- lifespan:`from fastmcp.server.lifespan import lifespan`;`@lifespan async def app_lifespan(server): ...; yield {"searcher": ...}`;`FastMCP("name", lifespan=app_lifespan)`;tool 里 `ctx.lifespan_context["searcher"]`

**langchain-mcp-adapters**(消费侧):
- `from langchain_mcp_adapters.client import MultiServerMCPClient`
- `client = MultiServerMCPClient({"name": {"command"/"url", "transport": "stdio"/"streamable-http", "args"}})`
- `tools = await client.get_tools()` → BaseTool 列表;全链路 async(`asyncio.run` + `ainvoke`);args 用**绝对路径**

**测试**(FastMCP in-memory):
- `async with Client(server) as c: await c.call_tool("search_notes", {...})`(同进程,毫秒级,不部署)
- 别在 fixture 里 open client(event loop 坑),在 test 内 `async with`

**hybrid**:
- `bm25s.BM25(corpus).index(jieba 分词)` → `retrieve(jieba(query), k)`
- **BM25 必须 chunk 级**(id 与 Chroma 对齐,否则 RRF 融合对不上)
- `RRF(d) = Σ 1/(60 + rank)`(~10 行)

## 5. 关键坑(Windows + 协议)

- **绝对路径**:MultiServerMCPClient 的 args、claude_desktop_config 都要绝对路径
- **async 生命周期**:client 在 `main()` 内存活,`get_tools()` 后别退 async 上下文
- **stdio 别 `print`**:server 用 `logging`(print 污染 JSON-RPC)
- **ollama IPv6**:用 `127.0.0.1` 不用 `localhost`
- **Windows 路径**:用 `pathlib`,别硬编码反斜杠
- **venv 里 npx 可见性**:filesystem/fetch server 要 npx,确认 PATH
- **resource 模板路径遍历**:`is_relative_to` 校验
- **Chroma 维度**:bge-m3 固定 1024,collection 名带维度(如 `notes_bge-m3_1024`)
